Projeto de API de Precificação de Casas

Link do repositório GitHub: https://github.com/kotjiac/precificacao-venda-casas-api

Para executar a aplicação:
1. Clone o repositório
2. Execute com Docker:

   docker build -t precificador .
   docker run -p 8000:8000 precificador

3. Acesse a documentação em: http://localhost:8000/docs

Mais detalhes disponíveis no README.md do projeto.
